O que é o OSM?

O OpenStreetMap (OSM) não é propriamente um mapa, mas uma base de dados georreferenciados.
A maior do mundo em dados abertos.

 

 

O OSM não é apenas um mapa

Os dados normalmente são exibidos sob a forma de mapas, através de layers TMS¹ renderizados do OSM para slippy-maps, na web: em browser (ao lado) ou mobile.

¹ Tile Map Service

Qual o Sistema de Coordenadas (CRS) do OSM?

  • Os dados: WGS84 (EPSG:4326)
     
  • As camadas TMS (mapas renderizados, imagens de satélite): Web Mercator (ou Pseudo-Mercator, EPSG:3857)

https://wiki.openstreetmap.org/wiki/Converting_to_WGS84.

LICENÇA DE USO DOS DADOS - DIREITOS AUTORAIS:

  • É livre o acesso, uso, difusão, adaptação, dos dados do OSM, sob licença ODbL 1.0, requerendo-se que os créditos constem visíveis.
  • A cartografia dos mapas, renderização, tiles e documentação, sujeitam-se à licença Creative Commons / Attribution-ShareAlike 2.0 Generic (CC BY-SA 2.0), requerendo-se que, em nova produção sobre ou usando o material, os créditos constem visíveis.
  • O modo de atribuir os créditos, eventualmente discriminando o que foi utilizado (se mapa base renderizado ou somente dados), é fazer constar em material publicado: © Contribuidores do OpenStreetMap

Desvantagens de mapas privados:

  • Faltam mapas de cidades pequenas ou áreas economicamente pouco desenvolvidas.
  • O foco do mapa geralmente é em informação para carros.
  • Demora em ser atualizado.
  • Não temos os dados brutos.
  • Não podemos editar o mapa.
  • Quando se pode editar, suas contribuições viram propriedade da empresa.

Quem pode adicionar os dados?

Funciona da mesma forma que a Wikipédia.

QUAIS FORMATOS DE ARQUIVOS O OSM DISPONIBILIZA?

  • .osm: xml format
  • .pbf: Protocolbuffer Binary Format (smaller to download and much faster to process)
  • .bz2: compressed
